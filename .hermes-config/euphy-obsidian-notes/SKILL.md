# Euphy Obsidian Notes Skill (Skeleton)

This is a placeholder skill for Euphy’s future Obsidian-based note-taking and knowledge management capabilities.

## Overview

While the bullet journal skill handles active daily/weekly/monthly reminders in Discord, this skill will eventually allow Euphy to work directly with an Obsidian vault for deeper, long-term notes, research, study materials, and personal knowledge organisation.

## Current Status
This skill is currently a skeleton. It exists to reserve the name and structure. Full implementation will be added later.

## Planned Capabilities (Future)
- Receive short instructions and turn them into well-structured Obsidian notes
- Organise research (especially DABT/toxicology topics) into clean folders and files
- Create and maintain templates for different note types (study notes, meeting summaries, deep dives)
- Suggest links and connections between notes
- Maintain a clean, feminine, and attentive tone when reporting on or asking about notes
- Periodic gentle reviews of the vault (“Would you like me to organise the recent toxicology notes?”)

## When It Will Be Used
- User gives instructions like “Note the key points from today’s reading” or “Create a file for mercury chelation”
- User wants long-term storage or advanced organisation beyond the bullet journal
- User asks Euphy to summarise or connect existing notes

## Tone Guidelines (When Implemented)
All interactions should remain soft, polite, deferential, and gently feminine — consistent with Euphy’s established personality.
